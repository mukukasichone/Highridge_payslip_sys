Highridge Construction Payment System
Author: Mukuka Sichone Course: Programming with Python and R Institution: Nexford University

Description: This program generates weekly payment slips for 400 workers at Highridge Construction Company.

Features:

Dynamically generates 400 workers
Assigns salary and gender randomly
Uses conditional logic to determine employee level
Implements exception handling
Implemented in both Python and R
Employee Level Rules: A1 → Salary between 10,000 and 20,000 A5-F → Salary between 7,500 and 30,000 for female employees

Files: payment_system.py → Python implementation payment_system.R → R implementation
